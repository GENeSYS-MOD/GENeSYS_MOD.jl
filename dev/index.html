GENeSYS_MOD.jl

This Julia package provides a Julia implementation of the General ENergy SYStem MODel, GENeSYS-MOD (originally in GAMS). GENeSYS_MOD.jl.

GENeSYS_MOD — Module

Main module for GENeSYS_MOD.jl.

This module provides the means to run GENeSYS-MOD in julia. It is a translation of the GAMS version of the model.
